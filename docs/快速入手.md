# 明察 MingCha —— 5 分钟快速入手

> 目标：从零到浏览器里分析第一个视频。照着敲即可，不需要懂命令行细节。
> 完整说明见 `前后端分离-启动指南.md`；本文只走**最短路径**。

---

## 你需要准备什么

1. **Python 3.10+** —— 命令行敲 `python --version` 能看到版本号即可。
2. **一个模型 Key** —— Claude / OpenAI(GPT) / GLM 任选其一，界面里填。
3. 视频 —— 本地文件或一条能下载的链接。

> ffmpeg 不用手动装，下面第 1 步的 `static-ffmpeg` 会自动搞定。

---

## 第 1 步：装依赖（只做一次）

打开终端，`cd` 到项目根目录 `mingcha/`，逐行执行：

```bash
# 建一个独立环境，避免污染系统 Python
python -m venv .venv

# 激活环境
.venv\Scripts\activate          # Windows
# source .venv/bin/activate     # Mac / Linux

# 装内核 + 后端 + ffmpeg + 视频下载器
pip install -e . -e ./backend
pip install static-ffmpeg "yt-dlp>=2024.1.0"
```

看到没有红色报错就成功了。

> 想让「上传的视频自动生成字幕」？再装一个（可选，体积较大）：
> `pip install openai-whisper`

---

## 第 2 步：起服务

```bash
uvicorn app.main:app --app-dir backend --port 8000
```

看到这行就说明起来了：

```
Uvicorn running on http://127.0.0.1:8000
```

**保持这个终端别关**。

---

## 第 3 步：打开浏览器

访问 👉 **http://localhost:8000**

你会看到明察的界面。

---

## 第 4 步：分析第一个视频

1. **填 Key**：点开「模型 Key 设置」，把你的 Key 填进对应的框（Claude / OpenAI / GLM 任选）。
   勾上「记住 Key」下次就不用再填（只存你自己电脑的浏览器里）。
2. **选视频**：上传本地文件，或切到「粘贴链接」贴一条视频 URL。
3. **写提问**，直接决定分析类型（也可手动选意图）：

   | 你想干嘛 | 就这么问 | 会用到 |
   |---|---|---|
   | 看视频讲了啥 | `总结这个视频讲了什么` | 理解摘要 |
   | 找某东西第一次出现 | `红色的车最早出现在什么时间` | 定位 |
   | 查有没有违规内容 | `有没有暴力/血腥内容` | 审核 |
   | 用一张图在视频里找 | 附上参考图 + `找到这个人出现的位置` | 以图搜 |

4. 点 **开始分析**，看进度条走完。
5. 出结果后：点任意**证据截图**，下方播放器会自动跳到那一秒。

---

## 常见卡点

| 现象 | 原因 / 解决 |
|---|---|
| `uvicorn: command not found` | 环境没激活，重跑第 1 步的「激活环境」那行 |
| 页面能开但一分析就报「模型调用失败」 | Key 没填 / 填错 / 没余额；换个 provider 的 Key 试试 |
| URL 视频下载失败 | 可能是私有视频，需要 cookies；先换个公开链接验证流程 |
| 结果提示「无字幕」 | 该视频没字幕且没装 whisper；装 whisper 或换有字幕的视频 |
| 想让局域网别的设备也能用 | 启动加 `--host 0.0.0.0`（注意：Key 和产物会对同网段可见，谨慎） |

---

## 只想用命令行？

不开浏览器也行，装完第 1 步后直接：

```bash
mc ask 视频.mp4 "总结这个视频"
```

更多 CLI 用法见 `docs/使用手册.md`。

## ⚠️ 重要偏差：完全自研，非「改 crv」

设计文档 §1.2 / §3 的原方案是「明察 `import` crv + 对 crv 做最小 G1 改动」。**实际按用户决策改为「完全自研、不依赖 crv」**：

- crv **一行未改**（仅保留它自身的 `-vsync→-fps_mode` 兼容修复，与明察无关）。
- 媒体 IO（取视频/转写/音频/探测）**移植进 `media.py`**，成为明察自有代码。
- 抽帧/去重/拼图 + 时间戳链路在 `preprocess.py` / `timestamps.py`。
- 明察是**单包独立项目**，`pip install -e .` 即可，无需装 crv。
- 已验证：屏蔽 `claude_real_video` 后全链路照常运行。
